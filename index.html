cloning repo

in this we used to create a folder by cloning created repo from github using git clone url command

then we have to add html and add in git by git add.

now we have to commit by git commit -m "changes made" to save the changes

then we can check status by git status

we can check logs by git log

we can check differences between previous version and present changes in code by git diff

finally we are going to push the folder into github by git push orgin main
